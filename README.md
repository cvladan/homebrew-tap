# homebrew-tap

Personal [Homebrew tap](https://docs.brew.sh/Taps) for [@cvladan](https://github.com/cvladan)'s casks.

## Usage

```sh
brew tap cvladan/tap
brew install --cask <cask-name>
```

## Casks

- **[browserpick](https://github.com/cvladan/browser-pick)** — tiny macOS menubar app that lets you pick which browser opens a link.

  ```sh
  brew install --cask cvladan/tap/browserpick
  ```

- **[moneyGuru](https://github.com/cvladan/moneyguru)** — personal finance management application for Apple Silicon Macs.

  ```sh
  brew install --cask cvladan/tap/moneyguru
  ```
